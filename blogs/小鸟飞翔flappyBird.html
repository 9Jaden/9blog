    主要要素：小鸟、管道以及动态画面

    小鸟【用div表示，且默认向下】：

    <div id="bird" class="birdDown"></div>
        .birdDown {
            animation: birdDown 0.05s linear infinite;
        }
        @keyframes birdDown {
            from {
                background-image: url(img/down_bird0.png);
            }
            to {
                background-image: url(img/down_bird1.png);
            }
        }
// keyframes动画是展示一定小鸟扑翅的效果

    管道：计数控制管道生成速度；管道分为上管道和下管道，用ul包含所有管道，里面的li代表管道，用class为top/bottom渲染上下管道以及不同height相同的left

            <ul>

    管道移动及生成：

    ​ 不断向左移动 pipe.style.left = pipe.offsetLeft - 2 + "px"

    ​ 如果移出屏幕【offsetLeft判断】则直接ul.removeChild移除该管道

    小鸟自动下坠：bird.style.top = bird.offsetTop + speed + "px"

    ​ 而速度speed为变量，取决于用户是否点击，点击后speed为负并不断变正【以更大速度+=0.7】，而speed>=0时下降表示为true，下坠加速度变小+=0.4

    碰撞检测：考虑因素---pipe和bird的offsetLeft/offSetTop以及offSetWidth和offsetHeight

    function crushPipe(pipe, bird) {
  let pLeft = pipe.offsetLeft; // 管道到屏幕左侧的距离
  let pTop = pipe.offsetTop;
  let pRight = pLeft + pipe.offsetWidth;
  let pBottom = pTop + pipe.offsetHeight;

  let bLeft = bird.offsetLeft; // 小鸟到屏幕左侧的距离
  let bTop = bird.offsetTop;
  let bRight = bLeft + bird.offsetWidth;
  let bBottom = bTop + bird.offsetHeight;

  if (bRight < pLeft || bLeft > pRight || bTop > pBottom || bBottom < pTop) {
    // 说明未碰撞
      // 四个条件同时不满足才说明碰撞
    return false;
  } else return true;
}

    通过管道计分：

    ​ 写在计时器不断调用的管道移动函数里，用pipe.offsetLeft + pipe.offsetWidth < 20判断，其中20为小鸟的left值
