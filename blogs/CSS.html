    # flex---flexible box

    Flex 是 Flexible Box 的缩写，意为"弹性布局"，用来为盒状模型提供最大的灵活性

    可以简便、完整、响应式地实现各种页面布局

    垂直居中：align-items: center

    水平居中：justify-content: center

    # flex: 1

    flex 属性是 flex-grow, flex-shrink 和 flex-basis 的简写，默认值为 0 1 auto。后两个属性可选。

    flex-grow 属性定义项目的放大比例，默认为0，即如果存在剩余空间，也不放大。如果所有项目的flex-grow属性都为1，则它们将等分剩余空间（如果有的话）。如果一个项目的flex-grow属性为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。

    flex-shrink 属性定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。如果所有项目的flex-shrink属性都为1，当空间不足时，都将等比例缩小。如果一个项目的flex-shrink属性为0，其他项目都为1，则空间不足时，前者不缩小。

    flex-basis属性定义了在分配多余空间之前，项目占据的主轴空间。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小。

    flex 属性属性有两个快捷值：auto (1 1 auto) 和 none (0 0 auto)。

    建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值。

    所以flex: 1表示的含义是等分剩余空间

    # flex实现两栏布局

    一栏固定宽度，一栏自适应

    .wrap { display: flex; }
.left { width: 200px; flex: 0 0 200px}
.right { flex: 1; }

    # css水平垂直居中

    水平

    行内元素 text-align: center;

    块级元素 margin: 0 auto

    flex弹性盒子 display: flex; justify-content: center;

    绝对定位(子绝父相) position: absolute; left: 50%; transfrom: translate(-50%, 0);

    垂直

    行内块元素 vertical-align: middle;

    flex弹性盒子 display: flex; align-items: center;

    绝对定位(子绝父相) position: absolute; top: 50%; transfrom: translate(, -50%);

    文字的话：行高line-height等于字体大小font-size

    # 盒模型及BFC

    标准盒模型box-sizing: content-box border margin padding content

    还有box-sizing: border-box

    标准盒与怪异盒的区别在于他们的总宽度的计算公式不一样。 标准模式下总宽度=width+margin（左右）+padding（左右）border（左右）；怪异模式下总宽度=width+margin（左右）（就是说width已经包含了padding和border值）所以在宽度一定的情况下，增大padding或者border挤压content的大小

    # BFC

    块级格式化上下文，可以看做隔离的独立容器，不会影响外部

    布局规则：内部块级盒子垂直排列；垂直方向距离由margin决定；同一个bfc内margin会重叠

    触发：

    display: inline-block/flex触发BFC

    绝对定位

    或者父子外边距合并时，父overflow: hidden触发BFC

    用途：清除浮动，解决外边距合并，父元素高度塌陷，布局

    # 清除浮动的方法

    浮动的目的：脱离文档流，把多个块级元素放到想要的一行上

    清除的目的：浮动会影响标准流，比如浮动造成的父级盒子塌陷。让父元素闭合出口和入口不让子浮动出来影响其他元素

    方法:

    1. <div style='clear: both'> </div> // 末尾添加空标签--->代码冗余

    2. overflow: hidden;

    3. 伪元素，只需写一次后在需要清除浮动的标签里加上clearfix类别即可

    .clearfix::after{
    content: '';
    display: block;
    clear: both;
}

    # 父级盒子塌陷

    如果父元素只包含浮动元素且未设置高度和宽度时，那么它的高度就会坍缩为0

    解决方法：

    1. 父添加高度；
    2. 父overflow:hidden;
    3. 父也添加浮动;
    4. 父设置为块级元素display:inline-block;
    5. 清除浮动：伪元素
    .clearfix::after{
    content: '';
    display: block;
    clear: both;
}

    转为BFC

        行内（不独占、不可设置宽高、自己撑开）---a、span标签：

        行内元素（内联元素）：不占有独立的区域，仅仅依靠自己的字体大小或者是图像大小来支撑结构。 一般不可以设置宽度，高度以及对齐等属性。 常见的行内元素有：,,,,,等 默认的宽度就是它本身的宽度

        行内块（可设置宽高，也可一行放多个）：

        块级元素，它既具有块级元素的特点，也有行内元素的特点，它可以自由设置元宽度和高度， 也可以在一行中放置多个行内块级元素。比如input、img就是行内块级元素，它可以设置高宽以及一行多个。

        # CSS样式选择器及优先级

        • ID 选择器， 如 #id{}
        • 类选择器， 如 .class{}
        • 属性选择器， 如 a[href="segmentfault.com"]{}
        • 伪类选择器， 如 :hover{}
        • 伪元素选择器， 如 ::before{}
        • 标签选择器， 如 span{}
        • 通配选择器， 如 *{}
        • 内联样式：

        **CSS 优先规则3：**优先级关系：内联样式 > ID 选择器 > 类选择器 = 属性选择器 = 伪类选择器 > 标签选择器 = 伪元素选择器

        # 伪类:hover和伪元素::after

        伪元素（两个冒号）其实相当于伪造了一个元素，而伪类（一个冒号）没有伪造元素，它用于选择处于特定状态的元素（添加特定样式如hover）

        # px em rem vw vh vm

        • px：绝对长度单位，代表屏幕中每个「点」( pixel )。
        • em：相对长度单位，每个子元素透过「倍数」乘以父元素的 px 值。
        • rem：相对长度单位，每个元素透过「倍数」乘以根元素的 px 值。

        通过rem做到只修改根元素就成比例地调整所有字体大小

        vw是相对视口（viewport）的宽度而定的，长度等于视口宽度的1/100

        vh是相对视口（viewport）的高度而定的，长度等于视口高度的1/100

        vm是相对于视口的宽度或高度中较小的那个，其中最小的那个被均分为100单位的vm

        # 移动端适配

        1. 需要缩放的元素使用rem或vw等相对单位，不需要缩放的使用px
        2. 固定尺寸+弹性布局，不需要缩放 display: flex;

        # 骰子三和五

        用flex布局实现

        注意会用 ul > li : n-th child(3)取到子节点

        justify-content: space-around; align-items: center 关键！！！

        以及flex-wrap: wrap 对于四五六筒，要换行

        但是三筒没有align-items 使用align-self做的，后两个分别为center和flex-end

        <div class="box">
      <div class="dot"></div>
      <div class="dot"></div>
      <div class="dot"></div>
</div>

<style>
      .box {
        width: 200px;
        height: 200px;
        background-color: #f1f1f1;
        display: flex;
        justify-content: space-around; /* 均匀排列每个元素每个元素周围分配相同的空间 */
      }
      .dot {
        width: 30px;
        height: 30px;
        background-color: black;
        border-radius: 50%; /* 做圆形 */
      }
      .dot:nth-child(2) {
        align-self: center;
      }
      .dot:nth-child(3) {
        align-self: flex-end;
      }
 </style>

        <ul class="MJContainer fiveCone">
   <li></li>
   <li></li>
   <li></li>
   <li></li>
   <li></li>
  </ul>

<style>
ul.fiveCone > li{/* 五筒 */
   margin: 0 10px;
}
ul.fiveCone > li:nth-child(3){
   margin: 0 30px;
}
ul.fiveCone{
   width: 30px;
   height: 30px;
   background-color: black;
   border-radius: 50%; /* 做圆形 */
   flex-wrap: wrap; /* 换行 */ 
   padding: 5px 0;
   justify-content: space-around;
   align-items: center;
}
</style>

        一筒居中即可：justify-content和align-items都为center

        二筒改变轴flex-direction: column; 后justify-content: space-around; align-items: center

        四筒和六筒注意flex-wrap: wrap (要换行的意思) 然后也是justify-content: space-around; align-items: center
